Kali Linux та Parrot OS

Kali Linux

  • Debian-based, розробник — OffSec
  • 600+ попередньо встановлених інструментів
  • Робоче середовище — XFCE
  • Rolling release — завжди актуальні версії
  • Метапакети: kali-linux-default, kali-tools-web, kali-tools-wireless
  • Non-root за замовчуванням з 2020 року

Parrot OS

  • Debian-based, легковагий дистрибутив
  • Робоче середовище — MATE
  • AnonSurf — вбудована анонімізація трафіку
  • Дві редакції: Security та Home
  • Фокус на приватність та анонімність
  • Менше споживання ресурсів

Еволюція

BackTrack → Kali Linux (2013) — перехід від CD-based дистрибутива до повноцінної rolling release системи з постійними оновленнями та модульною архітектурою.

Переваги Linux для пентесту

Відкритий код

Повний контроль над системою, можливість аудиту вихідного коду та модифікації під власні потреби.

Потужний термінал

Bash/Zsh, пайпи (|), перенаправлення, автоматизація задач та скриптинг.

Нативна підтримка

Більшість інструментів безпеки написані для Linux та працюють нативно без емуляції.

Підсумок

Головна ідея

Linux є стандартом для пентесту завдяки відкритому коду, потужному терміналу та нативній підтримці інструментів безпеки.

  • Kali Linux — найпопулярніший дистрибутив для пентесту
  • Parrot OS — легша альтернатива з фокусом на приватність
  • 600+ попередньо встановлених інструментів
  • Потужний термінал та скриптинг
