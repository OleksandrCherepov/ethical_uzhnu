APT — менеджер пакетів

Що таке APT?

APT (Advanced Package Tool) — високорівневий менеджер пакетів для Debian/Ubuntu/Kali. Автоматично вирішує залежності.

Оновлення системи

  • sudo apt update — оновити список пакетів
  • sudo apt full-upgrade — оновити пакети (для Kali!)
  • sudo apt autoremove — видалити непотрібне

Встановлення / видалення

  • sudo apt install nmap
  • sudo apt remove package
  • sudo apt purge package — з конфігами
  • apt search keyword
  • apt show package

Важливо

Для Kali Linux використовуйте full-upgrade, а не upgrade — rolling release потребує інтелектуального оновлення залежностей.

Метапакети Kali та systemctl

Метапакет Призначення Приклади інструментів
kali-linux-default Стандартний набір nmap, metasploit, burpsuite
kali-tools-web Веб-тестування sqlmap, nikto, wpscan
kali-tools-wireless Бездротові мережі aircrack-ng, wifite
kali-tools-passwords Атаки на паролі john, hashcat, hydra
kali-tools-exploitation Експлуатація metasploit, searchsploit

systemctl — керування сервісами

  • systemctl start ssh — запустити сервіс
  • systemctl stop apache2 — зупинити сервіс
  • systemctl enable ssh — увімкнути автозапуск
  • systemctl status postgresql — перевірити стан

Підсумок

Головна ідея

APT та метапакети дозволяють швидко розгорнути повноцінне середовище для пентесту.

  • apt update && apt full-upgrade — щотижневе оновлення Kali
  • apt install — встановлення інструментів
  • Метапакети — тематичні набори інструментів
  • systemctl — керування сервісами (SSH, PostgreSQL)
  • dpkg — робота з локальними .deb файлами
