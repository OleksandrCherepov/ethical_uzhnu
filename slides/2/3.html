Базова навігація та перегляд файлів

Навігація

pwd                 # поточний каталог
cd /path            # перехід за абсолютним шляхом
cd ~                # домашній каталог
cd -                # попередній каталог
cd ..               # на рівень вгору
ls -la              # детальний список + приховані
ls -lah             # з людиночитаним розміром
ls -R               # рекурсивний список

Перегляд файлів

cat file.txt        # вивести весь вміст
less file.txt       # посторінковий перегляд
head -n 20 file.txt # перші 20 рядків
tail -f /var/log/auth.log
# моніторинг логів у реальному часі

Порада

tail -f — незамінний інструмент для моніторингу логів під час пентесту.

Операції з файлами та пошук

Файлові операції

cp source dest
mv old new
rm -rf dir/
mkdir -p path/to/dir
touch file.txt

Пошук

find / -name "*.conf"
find / -perm -4000
# SUID-файли!
locate password.txt
which nmap
whereis python

Аналіз файлів

file document.pdf
# визначення типу файлу
strings binary
# витяг рядків з бінарника
stat file.txt
# метадані файлу

Важливо для пентесту

find / -perm -4000 2>/dev/null — одна з перших команд для пошуку привілейованих ескалацій.

Підсумок

Головна ідея

Ефективна навігація в терміналі — основа роботи пентестера.

  • pwd / cd / ls — базова навігація
  • cat / less / head / tail — перегляд вмісту
  • find з -perm -4000 — пошук SUID для привілеїв
  • strings — витяг тексту з бінарних файлів
  • Підстановочні символи: *, ?, [abc]

Далі

Наступна тема — система прав доступу та привілеї.
