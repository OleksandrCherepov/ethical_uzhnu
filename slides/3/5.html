theHarvester та Maltego

theHarvester

  • 38+ джерел даних для збору інформації
  • Збір email-адрес, субдоменів, IP-адрес, URL
  • Інтеграція з пошуковими системами та API
# Пошук по всіх джерелах theHarvester -d example.com -b all # Тільки Google theHarvester -d example.com -b google # LinkedIn профілі theHarvester -d example.com -b linkedin

Maltego

  • Візуальний аналіз зв'язків між сутностями
  • Три ключові концепції: Entities, Transforms, Graphs
  • Автоматичне збагачення даних через трансформації
  • Побудова графів зв'язків між людьми, доменами, IP

CE (Community Edition) — безкоштовна, обмежена кількість трансформацій. Pro — повний функціонал, комерційна ліцензія.

Shodan та Censys

Shodan

  • Пошукова система для IoT та підключених пристроїв
  • Збір банерів сервісів (HTTP, FTP, SSH, SMTP)
  • Виявлення вразливих систем та конфігурацій
port:3389 country:UA product:MySQL hostname:.gov.ua org:"Company Name" net:192.168.0.0/16

Censys

  • 100% покриття IPv4-простору за тиждень (vs 57% у Shodan)
  • Виявляє у 8 разів більше сервісів на хост
  • Глибокий аналіз TLS/SSL сертифікатів
  • Фокус на криптографічному стані Інтернету
Критерій Shodan Censys
Покриття ~57% IPv4 за тиждень ~100% IPv4 за тиждень
Швидкість Швидкий пошук, великий індекс Повільніший, глибше сканування
Фокус IoT, банери, вразливості Сертифікати, TLS, криптографія

Recon-ng, SpiderFoot та HIBP

Recon-ng

  • Інтерфейс у стилі Metasploit
  • Workspaces — ізоляція проєктів
  • Modules — recon, reporting, import, exploit
  • Вбудована БД для зберігання результатів
  • Marketplace для встановлення модулів

SpiderFoot

  • 100+ модулів збору даних
  • Автокореляція результатів між модулями
  • Зручний Web UI для візуалізації
  • Підтримка CLI та автоматизації
  • Інтеграція з зовнішніми API

Have I Been Pwned

  • Перевірка email на витоки даних
  • Pwned Passwords — база скомпрометованих паролів
  • API для автоматизованих перевірок
  • Synthient 2025: 1.9 млрд облікових записів у витоках

Підсумок

Екосистема OSINT-інструментів

Комбінація спеціалізованих інструментів дозволяє побудувати повну картину цілі — від доменів та IP до людей та витоків даних.

  • theHarvester та Maltego — збір та візуалізація зв'язків
  • Shodan та Censys — пошук пристроїв та сервісів в Інтернеті
  • Recon-ng та SpiderFoot — фреймворки автоматизації розвідки
  • Have I Been Pwned — перевірка витоків облікових записів
  • Кожен інструмент має свою нішу — використовуйте їх разом
