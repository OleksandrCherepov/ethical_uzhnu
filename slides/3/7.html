Веб-розвідка та метадані

Тема 3.7 — Розвідка та збір інформації (OSINT)

Кафедра ТЕІБ, Ужгородський національний університет

Фінгерпринтинг технологій

Wappalyzer

  • Браузерне розширення (Chrome, Firefox)
  • Виявляє 1000+ технологій у реальному часі
  • CMS, фреймворки, аналітика, CDN
  • Пасивний аналіз — без запитів до сервера
  • Категорії: JS-бібліотеки, платіжні системи, шрифти

BuiltWith

  • Пасивний онлайн-сервіс
  • Історія зміни технологій сайту
  • Конкурентний аналіз — порівняння стеків
  • Профілі технологій за галузями
  • API для масового аналізу

WhatWeb

  • CLI-інструмент, 900+ плагінів
  • Рівні агресії від 1 (пасивний) до 4 (heavy)
  • Виявляє CMS, сервери, фреймворки, помилки
whatweb -a 3 target.com

Wayback Machine та приховані файли

Wayback Machine

  • 866B+ збережених сторінок
  • Відновлення видаленого контенту
  • Старі сторінки "Про нас" — імена, структура
  • Попередні версії JS/CSS — приховані endpoint'и
  • Забуті піддомени та сервіси
waybackurls target.com | sort -u

Приховані файли та заголовки

  • robots.txt — список каталогів, які НЕ хочуть індексувати = найцікавіші для розвідки
  • sitemap.xml — повна структура сайту, всі URL
  • HTTP-заголовки:
Server: Apache/2.4.51
X-Powered-By: PHP/7.4.3

Версії ПЗ із заголовків → пошук відповідних CVE

Метадані документів

Інструменти

  • FOCA — Windows, автоматичний збір та аналіз метаданих
  • ExifTool — CLI, універсальний для будь-яких форматів
  • Metagoofil — пошук та завантаження документів з сайту
metagoofil -d target.com \
-t pdf,doc -l 100 -n 25

Що розкривають метадані

  • Імена користувачів — автори документів
  • Внутрішні шляхи: \\FILESERVER01\HR\
  • ПЗ: Microsoft Word 2019, Adobe Acrobat
  • Часові зони — географія офісів
  • Мережеві імена — принтери, сервери

Реальна загроза

Один PDF-файл з корпоративного сайту може розкрити 10+ внутрішніх username, структуру мережевих дисків та версії програмного забезпечення.

Підсумок

Головна ідея

Веб-розвідка та аналіз метаданих дозволяють отримати детальний профіль інфраструктури цілі без жодного активного сканування портів чи вразливостей.

  • Фінгерпринтинг технологій — Wappalyzer, BuiltWith, WhatWeb
  • Wayback Machine — відновлення видаленого контенту та старих версій
  • robots.txt, sitemap.xml, HTTP-заголовки — розкриття структури
  • Метадані документів — імена, шляхи, ПЗ, мережеві імена
  • FOCA, ExifTool, Metagoofil — автоматизація збору метаданих

Далі

У наступній темі розглянемо методологію розвідки та OPSEC — як систематизувати зібрану інформацію та залишатися непомітним.
