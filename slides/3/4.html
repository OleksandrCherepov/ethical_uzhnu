Оператори пошуку Google

Оператор Дія Приклад

filetype: Пошук файлів конкретного типу filetype:pdf security policy
intitle: Слово має бути у заголовку сторінки intitle:"index of" backup

GHDB — Google Hacking Database

Google Hacking Database від Offensive Security — колекція перевірених дорків для пошуку вразливостей: exploit-db.com/google-hacking-database. Категорії: файли з паролями, повідомлення про помилки, відкриті панелі, вразливе ПЗ.

Практичні дорки

Чутливі файли

  • filetype:sql "password"
  • filetype:env DB_PASSWORD
  • filetype:pdf "confidential"
  • filetype:log "password"
  • filetype:xls "username" "password"

Панелі адміністрування

  • intitle:"admin login"
  • inurl:wp-admin
  • intitle:"Dashboard [Jenkins]"
  • inurl:/phpmyadmin/
  • intitle:"Grafana" inurl:login

Directory Listings

  • intitle:"index of" password
  • intitle:"index of" .ssh
  • intitle:"index of" backup
  • intitle:"index of" .git
  • intitle:"index of" wp-content

Альтернативні пошуковики та правові аспекти

Альтернативні пошуковики

  • Bing — підтримує filetype:, site:, intitle:; інша індексація, інші результати
  • DuckDuckGo — підтримує базові оператори; не персоналізує результати
  • Yandex — потужна індексація пострадянського простору, оператор mime: замість filetype:

Різні пошуковики індексують різний контент — комбінуйте для повноти.

Правові межі

  • Легально: пошук публічної інформації через пошуковик
  • Легально: перегляд кешованих сторінок
  • НЕ легально: використання знайдених credentials для входу
  • НЕ легально: доступ до систем без дозволу власника

Увага

Google Dorking = легальний пошук, але використання знайдених даних (паролі, токени, ключі) для несанкціонованого доступу = злочин (ст. 361 КК України).

Підсумок

Головна ідея

Google Dorking — потужна техніка пасивної розвідки, що дозволяє знаходити чутливу інформацію без прямої взаємодії з ціллю, використовуючи лише пошукові системи.

  • Оператори site:, filetype:, intitle:, inurl: — основа Google Dorking
  • GHDB — готова база дорків від Offensive Security
  • Дорки знаходять файли, панелі, directory listings, credentials
  • Комбінування пошуковиків дає повнішу картину
  • Пошук легальний — несанкціонований доступ ні
