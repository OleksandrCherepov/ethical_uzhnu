Лекції Практичні Тести Матеріали

Практичні роботи

Закріпіть теоретичні знання на практиці. Кожна практична робота відповідає матеріалу відповідних лекцій.

01

Побудова лабораторії (Lab Setup)

До лекції 1

Нова

Розгортання віртуального середовища для проведення тестів на проникнення. Встановлення гіпервізора, налаштування Kali Linux та вразливої машини Metasploitable 2.

Завдання:

  1. Встановити VirtualBox або VMware Workstation Player
  2. Завантажити та розгорнути віртуальну машину Kali Linux
  3. Завантажити та розгорнути Metasploitable 2
  4. Налаштувати мережу Host-Only між Kali та Metasploitable
  5. Перевірити зв'язок між машинами командою ping
02

Основи Bash та скриптингу

До лекції 2

Скоро

Освоєння базових команд Linux-терміналу та створення простих Bash-скриптів для автоматизації рутинних завдань у сфері кібербезпеки.

Завдання:

  1. Виконати базові команди навігації: cd, ls, pwd, mkdir, rm
  2. Попрактикуватись з командами grep, cat, head, tail для роботи з файлами
  3. Написати Bash-скрипт, який приймає IP-адресу та виконує ping-сканування
  4. Розширити скрипт для збереження результатів у файл з міткою часу
03

Збір інформації про ціль

До лекції 3

Скоро

Практика пасивного та активного збору інформації (Reconnaissance) про цільову систему з використанням відкритих джерел та спеціалізованих інструментів.

Завдання:

  1. Виконати whois-запит для заданого домену та проаналізувати результати
  2. Використати nslookup / dig для отримання DNS-записів
  3. Запустити theHarvester для збору email-адрес та піддоменів
  4. Скласти звіт зібраної інформації про ціль (footprinting report)
04

Робота з Nmap

До лекції 4

Скоро

Глибоке сканування мережі за допомогою Nmap: визначення відкритих портів, версій сервісів, операційних систем та пошук потенційних вразливостей.

Завдання:

  1. Провести базове SYN-сканування Metasploitable 2 (nmap -sS)
  2. Виконати сканування версій сервісів (-sV) та визначення ОС (-O)
  3. Запустити скриптове сканування (-sC) для виявлення вразливостей
  4. Проаналізувати результати та визначити застарілі сервіси
  5. Зберегти результати сканування у форматах -oN та -oX
05

Використання Metasploit (System Hacking)

До лекції 5

Скоро

Практичне знайомство з Metasploit Framework: пошук експлойтів, налаштування параметрів атаки та отримання віддаленого доступу до вразливої системи.

Завдання:

  1. Запустити msfconsole та ознайомитися з базовими командами (search, use, info)
  2. Знайти експлойт для вразливого сервісу на Metasploitable 2
  3. Налаштувати параметри RHOST, LHOST та payload
  4. Виконати експлойт та отримати shell-доступ до цільової машини
  5. Продемонструвати виконання команд на зламаній системі
06

Злам сервісів методом перебору

До лекції 6

Скоро

Проведення атак методом грубої сили (Brute Force) на мережеві сервіси з використанням Hydra та допоміжних модулів Metasploit.

Завдання:

  1. Підготувати словники логінів та паролів для атаки
  2. Виконати brute-force атаку на SSH за допомогою Hydra
  3. Використати Metasploit Auxiliary модуль для підбору пароля до VNC
  4. Порівняти швидкість та ефективність Hydra та Metasploit Auxiliary
  5. Підключитися до зламаного сервісу з отриманими обліковими даними
07

SQL Injection (SQLi)

До лекції 7

Скоро

Практика виявлення та експлуатації SQL-ін'єкцій на веб-застосунку DVWA: від ручного тестування до автоматизованого дампу бази даних.

Завдання:

  1. Налаштувати DVWA на Metasploitable 2, встановити рівень безпеки Low
  2. Виконати ручну SQL-ін'єкцію через поле введення (UNION-based)
  3. Отримати список таблиць та стовпців бази даних вручну
  4. Використати SQLMap для автоматизованого виявлення SQLi
  5. Виконати дамп бази даних за допомогою SQLMap (--dump)
08

Веб-аналіз з Burp Suite

До лекції 8

Скоро

Перехоплення та аналіз HTTP-трафіку за допомогою Burp Suite. Налаштування проксі-сервера, модифікація запитів та пошук XSS-вразливостей.

Завдання:

  1. Налаштувати Burp Suite як проксі для браузера
  2. Перехопити HTTP-запити до DVWA та проаналізувати їх структуру
  3. Модифікувати перехоплений запит у вкладці Repeater
  4. Знайти та експлуатувати XSS-вразливість через Burp Suite
  5. Повторити перехоплення з OWASP ZAP (альтернативний інструмент)
09

Робота з сесіями Meterpreter

До лекції 9

Скоро

Поглиблена робота з Meterpreter: апгрейд простого shell до повноцінної сесії, збір даних, хешів паролів та знімків екрана зламаної системи.

Завдання:

  1. Отримати базовий shell через Metasploit та виконати апгрейд до Meterpreter
  2. Виконати команди sysinfo, getuid, ps для розвідки системи
  3. Зібрати хеші паролів за допомогою команди hashdump
  4. Зробити знімок екрана цільової машини (screenshot)
  5. Завантажити файл з цільової машини (download) та вивчити keyscan
10

Створення бекдору з msfvenom

До лекції 10

Скоро

Генерація шкідливих файлів (payload) за допомогою msfvenom та налаштування Multi/Handler для прийому зворотного з'єднання.

Завдання:

  1. Згенерувати payload у форматі .exe для Windows за допомогою msfvenom
  2. Згенерувати payload у форматі .apk для Android
  3. Налаштувати Multi/Handler у Metasploit для прийому з'єднання
  4. Протестувати зворотне з'єднання у лабораторному середовищі
  5. Дослідити параметри кодування payload для обходу антивірусів
11

Соціальна інженерія (Симуляція)

До лекції 11

Скоро

Симуляція атак соціальної інженерії за допомогою Social Engineering Toolkit (SET): клонування веб-сторінок та створення фішингових сценаріїв у контрольованому середовищі.

Завдання:

  1. Запустити Social Engineering Toolkit (SET) у Kali Linux
  2. Обрати Website Attack Vector та метод Credential Harvester
  3. Клонувати сторінку входу (наприклад, навчальний портал) у локальній мережі
  4. Протестувати збір облікових даних на клонованій сторінці
  5. Проаналізувати зібрані дані та підготувати звіт про вразливості
12

Підсумковий CTF (Capture The Flag)

До лекції 12

Скоро

Комплексне завдання Black Box: самостійний злам навчальної машини з використанням усіх вивчених технік та написання детального write-up звіту.

Завдання:

  1. Провести повну розвідку цільової машини (Reconnaissance)
  2. Виявити вразливості та визначити вектор атаки
  3. Експлуатувати знайдені вразливості для отримання початкового доступу
  4. Виконати ескалацію привілеїв (Privilege Escalation)
  5. Знайти всі прапорці (flags) та написати детальний write-up
