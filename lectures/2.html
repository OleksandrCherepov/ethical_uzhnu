Linux та командний рядок для хакера

2 год Нова

Огляд лекції

Linux є стандартною платформою для тестування на проникнення. У цій лекції ми розглянемо причини домінування Linux у сфері кібербезпеки, вивчимо ієрархію файлової системи, основні команди терміналу, систему прав доступу, управління пакетами, інструменти обробки тексту та мережеві утиліти. Особлива увага приділяється спеціалізованим дистрибутивам Kali Linux та Parrot OS, які містять попередньо встановлені інструменти для пентесту.

Студенти здобудуть практичні навички навігації файловою системою Linux, керування файлами та правами доступу, фільтрації тексту за допомогою grep, awk та sed, а також використання мережевих утиліт ssh, curl та netcat — усе це є фундаментом для подальшого вивчення етичного хакінгу та проведення тестувань на проникнення.

Теми лекції

  1. Чому Linux є стандартом для пентесту — Kali Linux, Parrot OS, переваги Linux, попередньо встановлені інструменти
  2. Файлова система Linux (FHS) — Ієрархія каталогів, ключові директорії для безпеки, файли конфігурації
  3. Навігація та робота з файлами — Команди cd, ls, find, cat, cp, mv, підстановочні символи
  4. Права доступу та користувачі — chmod, chown, SUID/SGID, Sticky Bit, привілеї
  5. Управління пакетами та сервісами — apt, dpkg, метапакети Kali, systemctl
  6. Фільтрація тексту та потоки — grep, awk, sed, пайпи, перенаправлення введення/виведення
  7. Мережеві утиліти та налаштування — ip, ss, curl, wget, ssh, netcat
